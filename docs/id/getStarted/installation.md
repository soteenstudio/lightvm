# Instalasi
Pilih salah satu metode instalasi di bawah ini sesuai dengan lingkungan proyek Anda:

## Dengan NPM
Gunakan metode ini jika Anda sedang membangun proyek berbasis **JavaScript/Node.js**.

```bash
npm install lightvm

# or
npm install lightvm@next
```

## Dengan Yarn
Alternatif bagi Anda yang lebih suka menggunakan **Yarn** sebagai pengelola paket.

```bash
yarn add lightvm

# or
yarn add lightvm@next
```

## Dengan Cargo
Jika Anda menggunakan **Rust**, gunakan **Cargo** untuk mengintegrasikan **LightVM** secara native.

```bash
cargo add lightvm

# or specific version
cargo add lightvm@0.1.0-alpha.6
```
