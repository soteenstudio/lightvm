# Security Policy

## Supported Versions

Security updates are prioritized for the latest active development releases. Please ensure you are using the most recent version before reporting a vulnerability.

| Version         | Supported          |
| --------------- | ------------------ |
| > 0.1.0-alpha.5 | :white_check_mark: |
| < 0.1.0-alpha.5 | :x:                |

## Reporting a Vulnerability

We take the security of our projects seriously. If you have discovered a security vulnerability, please do not disclose it publicly through issues or social media.

Instead, please report it directly and privately via the [GitHub Security Advisory](https://github.com/soteenstudio/lightvm/security/advisories/new?hl=id-ID).
